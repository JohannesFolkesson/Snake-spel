## About

This is a Snake game built with HTML5 Canvas and JavaScript. Players control a snake that grows longer as it eats food, avoiding collisions with walls and itself.

## Features

- Single-player mode
- Multiplayer mode (host and join sessions)
- Score tracking
- Sound effects
- Responsive canvas-based gameplay

## How to Play

1. Open `index.html` in a web browser
2. Click "Start Game" to begin
3. Use arrow keys to control your snake
4. Eat food to grow longer and increase your score
5. Avoid hitting walls or yourself!

## Multiplayer Mode

The game supports multiplayer gameplay:

1. One player clicks "Host Session" to create a game
2. Other players enter the Session ID and click "Join"
3. Players can compete to see who gets the highest score

## Files

- `index.html` - Main game page
- `app.js` - Application entry point
- `game.js` - Core game logic
- `snake.js` - Snake entity implementation
- `board.js` - Game board implementation
- `sounds.js` - Sound effects management
- `style.css` - Styling
- `mpapi.js` / `MultiplayerApi.js` - Multiplayer functionality
